# Go Statements and Basics

This document introduces core Go (Golang) concepts with corrected explanations and additional useful information.

---

## Variables

### Creating Variables (Explicit Declaration)

```go
var card string = "Ace of Spades"
```

Explanation:

* **var**: keyword used to declare a variable
* **card**: variable name
* **string**: variable type
* **"Ace of Spades"**: assigned value using `=`

You can also declare first and assign later:

```go
var card string
card = "Ace of Spades"
```

---

### Short Variable Declaration (Type Inference)

```go
card := "Ace of Spades"
```

Notes:

* `:=` automatically infers the variable type from the assigned value
* This syntax **can only be used inside functions**
* The variable must be **new** (not previously declared in the same scope)

## Basic Types

### Integers

#### Signed Integers

| Type    | Size               | Range                                                   |
| ------- | ------------------ | ------------------------------------------------------- |
| `int8`  | 8-bit              | −128 to 127                                             |
| `int16` | 16-bit             | −32,768 to 32,767                                       |
| `int32` | 32-bit             | −2,147,483,648 to 2,147,483,647                         |
| `int64` | 64-bit             | −9,223,372,036,854,775,808 to 9,223,372,036,854,775,807 |
| `int`   | platform-dependent | 32-bit or 64-bit                                        |

#### Unsigned Integers

| Type     | Size               | Range                           |
| -------- | ------------------ | ------------------------------- |
| `uint8`  | 8-bit              | 0 to 255                        |
| `uint16` | 16-bit             | 0 to 65,535                     |
| `uint32` | 32-bit             | 0 to 4,294,967,295              |
| `uint64` | 64-bit             | 0 to 18,446,744,073,709,551,615 |
| `uint`   | platform-dependent | 32-bit or 64-bit                |

#### Special Aliases

```go
byte // alias for uint8
rune // alias for int32 (represents a Unicode code point)
```

#### Important Note About `int`

* On **32-bit systems**: `int` is the same as `int32`
* On **64-bit systems**: `int` is the same as `int64`

## Functions

### Function Example

```go
func newCard() string {
	card := "Five of Diamonds"
	return card
}
```

Explanation:

1. **func**: keyword to declare a function
2. **newCard**: function name
3. **()**: function parameters (empty here)
4. **string**: return type
5. **return**: sends value back to the caller

### Function with Parameters

```go
func add(a int, b int) int {
	return a + b
}
```
### Function with multiple return value

```go
func addAndDivide(a int, b int) (int , int) {
	return a + b , a / b
}

add , divided := addAndDivide(4 , 2)
```

## Lists and Collections

### Arrays

Characteristics:

1. Fixed length
2. Same type for all elements

Example:

```go
var numbers [3]int = [3]int{1, 2, 3}
```

---

### Slices

Characteristics:

1. Dynamic size (can grow or shrink)
2. Same type for all elements
3. Most commonly used list type in Go

Example:

```go
cards := []string{"hi", "bye"}
```

Explanation:

* `[]`: indicates a slice
* `string`: type of elements
* `{}`: slice values

---

### Append to a Slice

```go
cards = append(cards, "how are you?")
```

Important notes:

* `append` **does not modify the original slice** directly
* It returns a **new slice**, so reassignment is required

---

### Split Slices

```go
cards := []string{"ai" , "backend" , "frontend"}

cards[0] // first element

cards[0:2] // 2 first elements

cards[-1] // last value
```

### Slice length

```go
cards := []string{"ai" , "backend" , "frontend"}

len(cards) // length of slice

```

### For Loop (range)

```go
for i, card := range cards {
	fmt.Println(i, card)
}
```

Or

```go
for i := 0; i < len(cards); i++ {
	fmt.Println(i, cards[i])
}
```

Explanation:

1. `for`: Go's loop keyword
2. `i`: index of the element
3. `card`: value of the element
4. `range`: iterates over the slice

Ignore index example:

```go
for _, card := range cards {
	fmt.Println(card)
}
```
